🐝 BeeTech E-commerce MVP
🚀 Sobre o Projeto

A BeeTech é uma loja especializada na venda de smartphones, acessórios e soluções tecnológicas que está expandindo sua atuação para o ambiente digital.

Este projeto tem como objetivo desenvolver um MVP (Minimum Viable Product) de um e-commerce moderno, seguro e escalável, permitindo que a BeeTech amplie suas vendas regionais e se prepare para operar em nível nacional.

🎯 Objetivos do Projeto

Desenvolver uma plataforma de e-commerce capaz de oferecer uma experiência de compra intuitiva, rápida e confiável.

Funcionalidades do MVP
✅ Rebranding completo da marca BeeTech
✅ Catálogo digital de smartphones e acessórios
✅ Pesquisa e filtros avançados
✅ Simulação de carrinho de compras
✅ Fluxo completo de checkout
✅ Persistência de dados em banco de dados
✅ Layout responsivo (Mobile First)
✅ Arquitetura escalável para futuras expansões
